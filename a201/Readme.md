ML: Machine Learning => Bütün kuralları tek tek yazmak yeriden veriden örüntü öğrenmesi sağlayan yaklaşımdır.
Klasik öğrenme: Kurallar+Veri = Sonuç
Makine öğrenme: Veri+ Sonuç örnekleri -> Model öğrenir -> Yeni sonuç üretir.

1- Sınıflandırma
2- Tahmin
3- Kümelemede
4- Karar destek
5- Öneri

Örnekler:
* Mail spam mı ? değil mi ?
* Kredi başvurusu riski ?
* Tahmini fiyatı
* Hangi İşlemde sahtekarlık ?
Veri topla -> veri temizle -> özellikler çıkar -> eğit -> test-> tahminde

A-) Denetimli öğrenme [Supervised Learning] (sınıfılandırma+regreyon)
* Özellikleri: Etiketlidir
 - Linear Regression
 - Logistic Regression
 - Decision Regression
 - Random Forest
 - KNN

B-) Denetimsiz öğrenme [Unsupervised Learning] 
* Özellikleri: Etiketsiz olması
- K-Means

C-) Reinforcement Learning

D-) Semi-supervised
- Tıbbi görüntülemede

(Görüntü işleme, Konuşma tanıma, Video analizi, doğal dil işleme)

Öğrenme mimarisi: 
1-CNN(görüntü verileri), 
2-RNN(Zaman serisi, sıralı veriler)
